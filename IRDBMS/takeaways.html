IRDBMS oefentoets take-aways

Een JOIN is zwaarder dan een subquery.

Triggers

Triggers kunnen gebruikt worden voor:

  • Notificaties
  • Integriteit onderhouden
  • Afgeleide data onderhouden
  • Views consistent houden
  • Business rules implementeren

Indexes en tuning

Indexes worden niet gebruikt bij sorteren.

Database tuning is niet belangrijk om de grootte van de database te beheersen.

Locking en concurrency

Conservative two-phase locking kan deadlocks voorkomen.

Wanneer een computer meerdere processoren heeft dan is parallel processing mogelijk. De meeste concurrency control theorieën gaan echter uit van interleaved processing.

Je kunt tijdens een transactie SAVEPOINT <naam> aangeven om terug te rollen naar dat punt.

Security

Deze stellingen zijn allebei juist:

  • Discretionary security mechanisme betreffen het toekennen en intrekken van rechten aan gebruikers.
  • Mandatory security mechanisme betreffen het doorvoeren van multi level security door data en gebruikers te classificeren.

Op account niveau (level) worden de volgende privileges verstrekt:

CREATE SCHEMA, ALTER, DROP, MODIFY.

Met RBAC kan de hiërarchie van verantwoordelijkheden in de organisatie goed worden weergegeven.
